## ðŸ“ž SUPPORT & TROUBLESHOOTING

### If Tests Fail:

1. **Check Environment:**
   - MongoDB running? (`mongod`)
   - Backend running? (`npm start` in /backend)
   - Frontend running? (`npm start` in /frontend)
   - All on correct ports? (27017, 5002, 3000)

2. **Check Data:**
   - Database initialized? Run seeding script if needed
   - Collections exist? Check MongoDB compass
   - Sample data present? Create event first

3. **Browser Issues:**
   - Clear cache (Ctrl+F5)
   - Try different browser
   - Check browser console for errors
   - Check network tab for API calls

4. **Specific Failures:**
   - Time formatting issues? Check inputFormatters.js
   - Cursor jump? Verify requestAnimationFrame in TimeInput
   - Lane mapping issues? Check heatGenerator.js IAAF_LANE_MAP
   - Database persistence? Query MongoDB directly
